# You have many Ways to Install pacflypy

Git Clone

```bash
git clone https://github.com/PacflyPy/pacflypy.git
cd pacflypy
pip install .
```

Git Archive

```bash
pip install --upgrade https://github.com/PacflyPy/pacflypy/archive/master.zip
```

PyPi

```bash
pip install --no-cache-dir pacflypy
```
